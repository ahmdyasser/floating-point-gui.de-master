Floating-point cheat sheet for Swift

Floating-Point Types

Almost all platforms map Swift floats to IEEE 754 double precision.

	let f = 0.1

Decimal Types

Swift has an arbitrary-precision decimal type named Decimal in the decimal module, which also allows to choose the rounding mode.

	let a = Decimal(0.1)
	let b = Decimal(0.2)
	let c = a + b # returns a Decimal representing exactly 0.3

How to Round

To get a string:

	String(format: "%.2f", 1.2399) # returns "1.24"
	String(format: "%.3f", 1.2399) # returns "1.240"
	String(format: "%.2f", 1.2399) # returns "1.20"

To print to standard output:

	print(String(format: "%.2f", 1.2399)) # just use print and String(format:_:...) method inside it

Specific rounding modesand other parameters can be defined with NumberFormatter().RoundingMode:

	let formatter = NumberFormatter()
	formatter.numberStyle = .decimal
	formatter.minimumFractionDigits = 2
	formatter.maximumFractionDigits = 2

print(formatter.string(from: 2.625)) // "2.62"
print(formatter.string(from: 2.635)) // "2.64"
